# ExpenseIQ – Smart Personal Expense Tracker

## Overview

ExpenseIQ is a full-stack web application that helps users track income and expenses, visualize spending trends, and manage personal finances through an intuitive dashboard.

## Features

• User Registration & Login
• Dashboard Summary
• Income & Expense Tracking
• Search & Filter Transactions
• Charts & Analytics
• Dark Mode
• Profile Management
• CSV Export

## Tech Stack

Frontend
- React
- Vite
- Axios
- React Router

Backend
- Spring Boot
- Java
- MySQL
- REST APIs

## Project Structure

expenseiq-frontend/
expenseiq-backend/

Frontend

Backend
